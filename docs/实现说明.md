## 文档维护规则

1. 每次完成一个功能实现后，必须新增一条记录。
2. 记录最少包含：背景、改动文件、操作步骤、功能说明、验证结果、后续事项。
3. 记录按时间倒序维护，最新记录放最上面。
4. 若本次未运行测试，必须明确写出原因。
